# Sudoku_Solver
This is a simple Sudoku Solver made in C++. It uses the Backtracking algorithm to solve a 9×9 Sudoku puzzle.
## Features
- Read puzzle from a text file
- Check if the puzzle is valid
- Solve the puzzle
- Show recursive calls and backtracks
- Save the solved puzzle
## How to Run
Compile:
g++ -std=c++17 src/main.cpp src/SudokuSolver.cpp -o SudokuSolver.exe
Run:
./SudokuSolver.exe
Enter the puzzle file name when asked, for example:
puzzles/easy.txt
